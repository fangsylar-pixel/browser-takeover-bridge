# Support

For installation questions and reproducible bugs, open a GitHub issue using the provided template.

Before reporting a problem:

1. Reload the unpacked extension from `chrome://extensions` or `edge://extensions`.
2. Open the popup and confirm the bridge status and synchronized tab count.
3. Use **Copy diagnostics** and remove any details you do not want to share.
4. Include the extension version, browser version, operating system, and exact reproduction steps.

For vulnerabilities or reports involving authenticated browser content, use the private process in
`SECURITY.md` instead of a public issue.
